# Architecture Decision Records (ADR)

This directory contains records of architectural decisions made in the OpenBricks project.

## Format

Each ADR follows this structure:

1. **Title**: Short descriptive name
2. **Status**: Proposed, Accepted, Deprecated, Superseded
3. **Context**: The issue or situation motivating the decision
4. **Decision**: The change or direction chosen
5. **Consequences**: Impact (positive and negative) of the decision
